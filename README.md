# react-tetris

React 俄罗斯方块，移动端优先，全手势拖拽操作。

----
### 操作方式

**全屏幕统一手势**（游戏开始后在棋盘上操作）：

| 手势 | 效果 |
|------|------|
| 水平滑动 | 左右移动方块 |
| 轻力下拉 | 加速下落（软降） |
| 大力下拉（≥60px） | 直落（硬降） |
| 点击 | 旋转（160ms 冷却） |

**游戏前**：点击顶部起始行/速度数字可调整参数（0-10 / 1-6），右侧喇叭图标控制音效开关。

----
### Fork 新增功能

本 Fork 在 [原版](https://github.com/chvin/react-tetris) 基础上增加了以下功能：

1. **全手势操作** — 移除物理按键区，棋盘上拖拽控制移动/下落，点击旋转。方向锁防止斜滑误触。
2. **幽灵方块** — 方块下落时底部显示灰色落点预览（#666）。
3. **Web Audio BGM** — 方波合成 Korobeiniki 主题曲，零文件体积。
4. **条件顶部栏** — 游戏前显示最高分/起始行/速度+音效开关；游戏中得分+消除行（左）、下一个（中）、重玩+音效（右）。
5. **紧凑布局** — 500px 容器、40px 格子、收窄边框间距。
6. **暂停遮罩** — 暂停时半透明遮罩覆盖棋盘，点击"继续"恢复。
7. **操作提示** — 开始按钮下方呼吸动画提示文字。
8. **方块直出** — 新方块直接出现在棋盘顶部，不再从上方探出。
9. **音效系统** — 统一音量控制（masterGain 0.7），点击调节参数有反馈音。

### 开发

#### 安装
```
npm install
```
#### 运行
```
npm start
```
浏览自动打开 [http://127.0.0.1:8080/](http://127.0.0.1:8080/)

#### 多语言
在 `i18n.json` 配置多语言环境，使用 `lan` 参数匹配语言如：`/?lan=en`

#### 打包编译
```
npm run build
```
构建产物在 `docs/` 目录。跨平台兼容（使用 rimraf + Node fs）。

#### 技术栈
- React 15 + Redux
- Immutable.js
- Webpack 3 / Babel
- Less (CSS Modules)
- Web Audio API
- CSS `transform: scale()` 响应式缩放
